# cr-lucky-chest-probability-calculator
Calculating the probability of lucky chests of different rarities

使用说明:
1. 设置宝箱等级数量和初始等级，点击"应用设置"
2. 设置各级升级概率(百分比)和升级次数
3. 点击"计算概率"按钮查看结果
4. 结果将显示各等级宝箱的概率和百分比
5. 右侧图表直观展示概率分布

注意:
- 升级概率是当前等级升级到下一等级的概率(百分比)
- 最高等级没有升级概率（无法继续升级）
- 概率值范围为0-100
- 初始等级必须小于等于总等级数
